# ✈️ Bonvo

**Your AI-powered travel companion.**
_An AI-powered travel assistant that crafts personalized itineraries, finds optimized flights/hotels, and provides local language guides based on your unique interests._

---

## 📌 Sorun (Problem)

Tatil planlamak streslidir. Uçuş, otel, gezi planı ve dil hazırlığı farklı araçlara dağılmıştır; üstelik bu araçların hiçbiri kullanıcının gerçek ilgi alanlarına göre kişiselleştirme sunmaz.

## ✨ Çözüm (Solution)

**Bonvo**, seyahat planlama sürecini uçtan uca **tek bir akışta** çözer:

- **Akıllı Profil:** İlgi alanları (Gastronomi ☕, Sanat 🎨, Fotoğrafçılık 📸) ve bütçe analizi.
- **Dinamik Rota:** Geniş ölçekli destinasyonlar için akıllı, çok şehirli rota önerileri.
- **Optimize Uçuş:** Fiyat bazlı listeleme ve veri odaklı akıllı tarih önerileri.
- **AI Plan (Streaming):** Kullanıcının hobilerine özel, gerçek zamanlı akan seyahat programı.
- **Dil Rehberi:** Gidilecek yerin yerel dilinde temel iletişim kalıpları.

## 🛠 Teknik Stack

| Katman             | Teknoloji                    |
| :----------------- | :--------------------------- |
| **Frontend**       | React 18 + TypeScript + Vite |
| **Backend**        | Java 17+ & Spring Boot 3     |
| **Styling**        | Tailwind CSS                 |
| **AI Integration** | Spring AI (Anthropic/OpenAI) |
| **Database**       | PostgreSQL / H2 (for MVP)    |
| **API Docs**       | Swagger / OpenAPI            |

## 🛡️ Veri Gizliliği

Kullanıcı verileri anonim tutulur ve yalnızca planlama amacıyla işlenir. GDPR/KVKK standartlarına uygundur.
